
# My Personal Portfolio

Welcome to my personal portfolio page! This site showcases my work, experience, and projects. Built to be minimalist, accessible, and fully responsive, it reflects my passion for clean design and modern web development.

## ✨ Features

- Minimalist, clean design
- Mobile-first and responsive
- SEO-friendly and accessible
- Easy to customize

## 🚀 Getting Started

To run this portfolio locally:

```bash
pnpm install
pnpm dev
```

Then visit [http://localhost:4321](http://localhost:4321) in your browser.
